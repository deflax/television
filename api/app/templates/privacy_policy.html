Privacy Policy

Last updated: {{ now.strftime('%Y-%m-%d') }}

This service provides live video streaming and optional archive access. We collect only the data needed to operate streaming, security checks, and reliability monitoring.

Information we process

  • Technical request data such as IP address, user agent, and access timestamps.
  • Session cookie data when you authenticate for archive access.
  • Timecode request events, including an obfuscated hostname label.

How we use information

  • To deliver the stream and related metadata (EPG, playlists, thumbnails, recordings).
  • To protect archive access and prevent abuse.
  • To monitor service health, active viewers, and operational performance.

Third-party services

When timecode requests are enabled, notifications can be sent to a Discord channel managed by the service operator.

Data security

Connections are intended to run over HTTPS, and access is restricted where required.

Contact

For privacy questions, contact the service operator for this domain.
