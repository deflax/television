## Full description

Deflax TV is a simple Android TV app for watching the live Deflax stream with an on-screen EPG.

Features:
- Live HLS playback optimized for TV
- Built-in electronic program guide (EPG)
- Current and next program display with countdown
- Remote-friendly controls to show/hide guide overlay
- Automatic stream reconnect on playback errors

Requirements:
- Internet connection
- Access to the Deflax stream endpoint
